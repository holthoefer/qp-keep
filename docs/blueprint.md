# **App Name**: Keep-Know

## Core Features:

- User Authentication: Secure user authentication via login screen.
- Note Management: Create, edit, and delete notes.
- Note Display: Display notes in a clean and organized manner.
- Intelligent Tagging: Suggest relevant tags based on note content. The LLM acts as a tool and has to reason when a given piece of information is appropriate in the output.
- Tag Persistence: Save tags together with note content.

## Style Guidelines:

- Primary color: A muted purple (#9467BD) to inspire focus.
- Background color: Light gray (#F0F0F5) to provide a comfortable contrast to the text.
- Accent color: Soft teal (#73C2FB) to highlight important actions or elements.
- Body font: 'Inter', a grotesque-style sans-serif font for readability and a modern feel.
- Headline font: 'Space Grotesk', a sans-serif with a modern touch for headers and titles.
- Simple, outlined icons for common actions.
- Clean, minimalist layout with plenty of whitespace to improve readability.
